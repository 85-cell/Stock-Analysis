📈 Stock Market Dashboard Web App

A dynamic Stock Market Dashboard built using HTML, CSS, and JavaScript that provides users with an interactive interface to explore stock-related insights such as live prices, top gainers/losers, news, and more.

This project focuses on creating a multi-page financial UI system with authentication flow and rich media integration.

🚀 Key Features
🔐 User Authentication System
Login & logout functionality (login.html, loginScript.js)
📊 Dashboard Interface
Central hub (index.html) for navigating stock features
📈 Live Stock Tracking
Dedicated page for viewing stock prices (liveStock.html)
🏆 Top Gainers & Losers
Market trend insights (gainersLosers.html)
🔎 Stock Search Functionality
Search UI for stocks (search.html)
📰 Stock Market News Section
News page with media support (news.html)
⚙️ User Settings Panel
Settings management (settings.html)
⏱️ Real-Time Clock Integration
JavaScript-based clock (clock.js)
🎥 Rich Media UI
Background videos and stock-related media for enhanced UX
🎨 Custom UI Design
Sidebar navigation (sidebar.css)
Login styling (loginStyle.css)
Responsive layout with images and branding assets
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Styling: Custom CSS (no frameworks)
Scripting: Vanilla JavaScript
Media: Images & video assets for UI enhancement
